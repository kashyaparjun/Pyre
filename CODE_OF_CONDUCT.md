# Code of Conduct

## Our standard

Participation in the Pyre project should be respectful, constructive, and professional.

Examples of expected behavior:

- showing empathy and respect
- giving actionable technical feedback
- assuming good faith while discussing design tradeoffs
- focusing criticism on ideas, code, and behavior rather than people

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- discriminatory language or behavior
- publishing private information without permission
- disruptive trolling or bad-faith engagement

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or contributions
that violate this Code of Conduct.

## Reporting

Report conduct issues to the project maintainer through the repository contact channel.
